Summary

Arise Admin Dashboard is a centralized space that highlights the most important tools used by admins. The special pages are built into it to make the Admin Dashboard the go-to place for admins and founders to easily tweak and monitor their community. The admin control panel is the area where administrators can log in and make changes affecting the functionality of the site, edit templates, add new staff and many other administrative functions. It is set up much like the staff control panel the main difference in what functionality is exposed to the logged in user.


Arise Admin dashboard includes admin page, custom pages and components, integration samples with 3rd party plugins, Profile, Invoice, Tables, Tasks, Cards, Users, Projects, Calendar, Widgets, C3 Graphs, D3 Graphs, Per Circle Graphs, Peity Graphs, jVector Maps and Wysiwyg Editor.
